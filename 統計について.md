# ② 平均（いちばん基本）

複数回測ったら、まずは平均をとる：

[
\text{平均} = \frac{x_1 + x_2 + x_3 + \cdots}{n}
]

👉 「だいたいこの辺の値だよね」という代表値

---

# ③ ばらつき（超重要）

平均だけだとダメです。

例：

* A：全部ほぼ同じ値
* B：バラバラ

→ 平均が同じでも信頼度が違う

そこで使うのが👇

## ● 標準偏差

[
\sigma = \sqrt{\frac{1}{n}\sum (x_i - \text{平均})^2}
]

👉 意味：
**データがどれくらい散らばっているか**

---

# ④ 誤差の考え方（ここが物理っぽい）

測定値はこう書きます：

[
L = 10.2 \pm 0.1 \ \text{cm}
]

👉 意味：

* 真の値はだいたいこの範囲にある

---

# ⑤ ランダム誤差 vs 系統誤差

## ● ランダム誤差

バラつく誤差（統計で処理できる）

* 人のブレ
* ノイズ

👉 平均すると減る

---

## ● 系統誤差

ズレ続ける誤差

* 定規が曲がってる
* ゼロ点がズレてる

👉 平均しても消えない

# ⑦ 正規分布（ベル型）

多くの測定はこうなる👇

* 真ん中に集まる
* 両端は少ない

👉 これを「正規分布」という

特徴：

* 約68% → ±1σ
* 約95% → ±2σ

---

# ⑧ まとめ（超重要）

物理実験の統計はこれ👇

* 平均 → 代表値
* 標準偏差 → ばらつき
* 誤差 → 信頼性
* グラフ → 理論との比較

👉 一番大事な考え方：

**「1回の測定ではなく、分布で考える」**

---

誤差伝播は一言でいうと、

**測定値に誤差があると、その値から計算した量にも誤差が伝わる**

という話です。

---

# 1. 問題設定

ある量

[
y=f(x)
]

を考えます。

測定値 (x) には誤差があり、

[
x = \bar{x}+\Delta x
]

と書けるとします。

ここで

* (\bar{x})：平均値・代表値
* (\Delta x)：小さな誤差
* (y=f(x))：計算したい量

です。

---

# 2. テイラー展開する

[
f(\bar{x}+\Delta x)
]

を (\bar{x}) のまわりでテイラー展開します。

[
f(\bar{x}+\Delta x)
===================

f(\bar{x})
+
\frac{df}{dx}\Delta x
+
\frac{1}{2}\frac{d^2f}{dx^2}(\Delta x)^2
+\cdots
]

誤差 (\Delta x) が十分小さいなら、2次以上を無視できます。

[
f(\bar{x}+\Delta x)
\approx
f(\bar{x})
+
\frac{df}{dx}\Delta x
]

したがって、

[
\Delta y
========

f(\bar{x}+\Delta x)-f(\bar{x})
\approx
\frac{df}{dx}\Delta x
]

つまり

[
\boxed{\Delta y \approx \left|\frac{df}{dx}\right|\Delta x}
]

これが**1変数の誤差伝播公式**です。

---

# 3. 複数変数の場合

次に

[
y=f(x_1,x_2,\dots,x_n)
]

を考えます。

それぞれの測定値に誤差があるとして、

[
x_i=\bar{x_i}+\Delta x_i
]

とします。

多変数テイラー展開すると、

[
\Delta y
\approx
\frac{\partial f}{\partial x_1}\Delta x_1
+
\frac{\partial f}{\partial x_2}\Delta x_2
+
\cdots
+
\frac{\partial f}{\partial x_n}\Delta x_n
]

つまり

[
\Delta y
\approx
\sum_{i=1}^{n}
\frac{\partial f}{\partial x_i}\Delta x_i
]

ここまでは「ズレそのもの」の伝播です。

---

# 4. なぜ二乗和になるのか

実験では、誤差はプラスにもマイナスにも揺れます。

だから単純に

[
\Delta y
========

\frac{\partial f}{\partial x_1}\Delta x_1
+
\frac{\partial f}{\partial x_2}\Delta x_2
]

と足すと、たまたま打ち消し合ってしまうことがあります。

そこで、分散を考えます。

[
\sigma_y^2
==========

\langle(\Delta y)^2\rangle
]

ここで

[
\Delta y
\approx
\sum_i
\frac{\partial f}{\partial x_i}\Delta x_i
]

なので、

[
\sigma_y^2
==========

\left\langle
\left(
\sum_i
\frac{\partial f}{\partial x_i}\Delta x_i
\right)^2
\right\rangle
]

展開すると、

[
\sigma_y^2
==========

\sum_i
\left(\frac{\partial f}{\partial x_i}\right)^2
\langle(\Delta x_i)^2\rangle
+
2\sum_{i<j}
\frac{\partial f}{\partial x_i}
\frac{\partial f}{\partial x_j}
\langle\Delta x_i\Delta x_j\rangle
]

ここで

[
\langle(\Delta x_i)^2\rangle=\sigma_i^2
]

です。

---

# 5. 独立な誤差なら共分散が消える

もし (x_i) と (x_j) の誤差が独立なら、

[
\langle\Delta x_i\Delta x_j\rangle=0
]

なので、

[
\boxed{
\sigma_y^2
==========

\sum_i
\left(
\frac{\partial f}{\partial x_i}
\right)^2
\sigma_i^2
}
]

したがって、

[
\boxed{
\sigma_y
========

\sqrt{
\sum_i
\left(
\frac{\partial f}{\partial x_i}
\right)^2
\sigma_i^2
}
}
]

これが高校物理・大学初年級でよく使う**誤差伝播の公式**です。

---

# 6. 相関がある場合の完全形

独立でない場合は、共分散を残します。

[
\boxed{
\sigma_y^2
==========

\sum_i
\left(
\frac{\partial f}{\partial x_i}
\right)^2
\sigma_i^2
+
2\sum_{i<j}
\frac{\partial f}{\partial x_i}
\frac{\partial f}{\partial x_j}
\operatorname{Cov}(x_i,x_j)
}
]

これがより一般的な形です。

---

# 7. 例：面積 (S=ab)

長方形の面積

[
S=ab
]

で、(a,b) に誤差があるとします。

偏微分は

[
\frac{\partial S}{\partial a}=b
]

[
\frac{\partial S}{\partial b}=a
]

だから

[
\sigma_S
========

\sqrt{
b^2\sigma_a^2
+
a^2\sigma_b^2
}
]

両辺を (S=ab) で割ると、

[
\frac{\sigma_S}{S}
==================

\sqrt{
\left(\frac{\sigma_a}{a}\right)^2
+
\left(\frac{\sigma_b}{b}\right)^2
}
]

つまり、掛け算では**相対誤差**が二乗和で足されます。

---

# 8. 例：速度 (v=\frac{x}{t})

[
v=\frac{x}{t}
]

偏微分は

[
\frac{\partial v}{\partial x}=\frac{1}{t}
]

[
\frac{\partial v}{\partial t}=-\frac{x}{t^2}
]

したがって、

[
\sigma_v
========

\sqrt{
\left(\frac{1}{t}\sigma_x\right)^2
+
\left(\frac{x}{t^2}\sigma_t\right)^2
}
]

相対誤差で書くと、

[
\boxed{
\frac{\sigma_v}{v}
==================

\sqrt{
\left(\frac{\sigma_x}{x}\right)^2
+
\left(\frac{\sigma_t}{t}\right)^2
}
}
]

割り算でも、相対誤差は二乗和で足されます。

---

# 9. まとめ

誤差伝播の本質はこれです。

[
\boxed{
\text{小さな入力誤差は、微分によって出力誤差に変換される}
}
]

つまり、

[
\Delta y
\approx
\frac{\partial f}{\partial x}\Delta x
]

です。

そしてランダム誤差では、符号がバラバラなので、単純な足し算ではなく

[
\boxed{
\sigma_y
========

\sqrt{
\sum_i
\left(
\frac{\partial f}{\partial x_i}
\right)^2
\sigma_i^2
}
}
]

になります。
