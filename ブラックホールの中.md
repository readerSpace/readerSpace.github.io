# まとめ

弦理論で相互作用が面積積分になるとは：

```text
点粒子：世界線の理論
弦　　：世界面の理論
```

ということです。

そして相互作用は、

```text
点粒子：一点でぶつかる
弦　　：世界面がなめらかにつながる
```

だから、短距離の発散がやわらぎます。
