**クラドニ図形（Chladni patterns）**とは、
振動している板の上に砂をまくと、**特定の模様が浮かび上がる現象**です。

18世紀の物理学者
👉 エルンスト・クラドニ
が発見しました。

---

## 何が起きているの？

ポイントは「振動」と「動かない場所」です。

### ① 板を振動させる

金属板などを弓やスピーカーで振動させると、
板全体がブルブル揺れます。

### ② でも全部が同じように動くわけじゃない

実は板には

* よく動く場所（振幅が大きい）
* **まったく動かない場所（節：node）**

ができます。

---

### ③ 砂が動く → 集まる

* よく動く場所 → 砂が跳ね飛ばされる
* 動かない場所（節） → 砂がたまる

👉 その結果、**線や模様が見える！**

---

## どうしてそんな模様になるの？

これは「波」の性質です。

板の中では振動が波として伝わり、
互いに重なり合って

👉 **定常波（standing wave）**ができます

---

### 定常波のイメージ

* 波がぶつかって固定されたパターンになる
* ギターの弦と同じ仕組み

---

## 数学的には？

板の振動は実は

👉 **波動方程式**で表されます

\nabla^2 u(x,y) = -\lambda u(x,y)

* (u(x,y))：板の振れ方
* (\lambda)：振動のモード（周波数に対応）

この解によって
👉 **模様（節の形）が決まる**

---

## なぜ模様が変わるの？

次の条件で変わります：

* 板の形（円・四角など）
* 固定の仕方（端を押さえる場所）
* 振動の周波数

👉 周波数を変えると
**全く違う幾何学模様が出る！**

---

## 直感まとめ

クラドニ図形はこう考えると理解しやすい：

* 板の中で「波」が立つ
* 波が打ち消し合う場所＝動かない
* そこに砂が集まる
* → 模様になる

---

## 面白いポイント

* 物理（波）と数学（固有値問題）がつながる
* 音（周波数）が「形」になる
* 実験がめちゃくちゃ視覚的でわかりやすい

---

## 発展（興味あれば）

* ドラムの膜の振動（太鼓）
* 量子力学の波動関数との類似
* 建築や音響設計への応用

以下を `chladni.html` として保存して、ブラウザで開くと動きます。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8" />
  <title>クラドニ図形シミュレーション</title>
  <style>
    body {
      margin: 0;
      background: #111;
      color: white;
      font-family: sans-serif;
      text-align: center;
    }
    canvas {
      background: #000;
      display: block;
      margin: 20px auto;
      border: 1px solid #555;
    }
    input {
      width: 300px;
    }
  </style>
</head>
<body>
  <h1>クラドニ図形</h1>

  <p>
    m: <input id="m" type="range" min="1" max="12" value="4">
    <span id="mVal">4</span>
  </p>
  <p>
    n: <input id="n" type="range" min="1" max="12" value="7">
    <span id="nVal">7</span>
  </p>

  <canvas id="canvas" width="600" height="600"></canvas>

  <script>
    const canvas = document.getElementById("canvas");
    const ctx = canvas.getContext("2d");

    const mSlider = document.getElementById("m");
    const nSlider = document.getElementById("n");
    const mVal = document.getElementById("mVal");
    const nVal = document.getElementById("nVal");

    const W = canvas.width;
    const H = canvas.height;

    function chladni(x, y, m, n) {
      // x, y は -1 ～ 1 の範囲
      return (
        Math.sin(m * Math.PI * x) * Math.sin(n * Math.PI * y) -
        Math.sin(n * Math.PI * x) * Math.sin(m * Math.PI * y)
      );
    }

    function draw() {
      const m = Number(mSlider.value);
      const n = Number(nSlider.value);

      mVal.textContent = m;
      nVal.textContent = n;

      const image = ctx.createImageData(W, H);
      const data = image.data;

      for (let py = 0; py < H; py++) {
        for (let px = 0; px < W; px++) {
          const x = (px / W) * 2 - 1;
          const y = (py / H) * 2 - 1;

          const v = chladni(x, y, m, n);

          // v が 0 に近いところが「節」
          const brightness = Math.exp(-Math.abs(v) * 40) * 255;

          const i = (py * W + px) * 4;
          data[i] = brightness;
          data[i + 1] = brightness;
          data[i + 2] = brightness;
          data[i + 3] = 255;
        }
      }

      ctx.putImageData(image, 0, 0);
    }

    mSlider.addEventListener("input", draw);
    nSlider.addEventListener("input", draw);

    draw();
  </script>
</body>
</html>
```

白い線が、砂が集まる「節」の部分です。
`m` と `n` を変えると、振動モードが変わって模様も変わります。
