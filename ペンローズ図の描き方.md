ペンローズ図（Penrose diagram）は、**時空の「因果構造」だけを保ったまま、無限遠を有限に押し込めて描く図**です。
ブラックホールや宇宙の構造を理解するための最強ツールです。

ここでは「描き方」を**直感 → 数学 → 手順**の順で整理します。

---

# ① まず本質（何をしているのか）

ペンローズ図は一言でいうと：

👉 「光の進み方（45度）を保ったまま、時空を縮める」

です。

* 光は常に斜め45°に進む
* 因果関係（行ける・行けない）だけ残す
* 距離や時間の長さは捨てる

この「因果構造」はGRの核心で、
ブラックホールの理解にも直結します。

---

# ② スタート：普通の時空図（ミンコフスキー）

まずは特殊相対論の時空：

[
ds^2 = -dt^2 + dx^2
]

これはミンコフスキー時空です。
 にあるように、この時空では光は

[
t = \pm x
]

つまり45°で進みます。

---

## 図のイメージ（通常）

```
      t
      ↑
      |
   \  |  /
    \ | /
----- O ------→ x
    / | \
   /  |  \
```

---

# ③ ステップ1：光座標に変換（超重要）

まず「光の方向」を基準にする

[
u = t - x \quad（左向き光）
]
[
v = t + x \quad（右向き光）
]

👉 これで光は

* u = const
* v = const

になる

---

# ④ ステップ2：無限を有限に押し込める（核心）

ここがペンローズ図の本質

例えば：

[
U = \arctan(u), \quad V = \arctan(v)
]

👉 すると

* u = ±∞ → U = ±π/2
* v = ±∞ → V = ±π/2

つまり
**無限遠が有限の点になる**

これは コンフォーマル変換 と呼ばれます。

 にあるように、ペンローズ図はこの「コンパクト化」に基づいています。

---

# ⑤ ステップ3：元の座標に戻す

[
T = \frac{U + V}{2}, \quad X = \frac{V - U}{2}
]

👉 これがペンローズ座標

---

# ⑥ ステップ4：図を描く

結果：

* 光：常に45°
* 全空間：有限領域に収まる

---

## ミンコフスキーのペンローズ図

```
        未来無限大 (i⁺)
          ▲
         / \
        /   \
       /     \
  左光無限   右光無限
 (𝓘⁻)     (𝓘⁺)
       \     /
        \   /
         \ /
          ▼
       過去無限大 (i⁻)
```

---

# ⑦ 境界の意味（超重要）

ペンローズ図の各点には意味がある：

| 記号              | 意味                  |
| --------------- | ------------------- |
| (i^+)           | 時間的未来無限（止まってる物体の終点） |
| (i^-)           | 時間的過去無限             |
| (\mathscr{I}^+) | 光が行く未来（光的無限）        |
| (\mathscr{I}^-) | 光の出発点               |

---

# ⑧ ブラックホールの場合

ペンローズ図を使うと：

* 事象の地平線が直線で描ける
* 特異点が「未来方向」に現れる

👉 これが
「未来が特異点に向く」
という話の正体

---

# ⑨ 描き方まとめ（実践手順）

### 手順

1. 計量を書く
2. 光座標 (u, v) を導入
3. (\arctan) などで有限化
4. 座標変換して描く
5. 光を45°で描く
6. 境界を解釈する

---

# ⑩ 直感まとめ（重要）

ペンローズ図とは：

👉 「時空の地図」ではなく
👉 「因果関係の地図」

---
いいですね、ここから一気に理解が深まるところです。
「手で描ける練習問題」を**レベル別に3段階**で出します。
全部やると、ブラックホールのペンローズ図まで自然につながります。

---

# 🟢 レベル1：ミンコフスキー時空（基本）

（最重要・まずこれ）

対象：ミンコフスキー時空

---

## 🔹問題1

次の手順でペンローズ図を描け：

1. 通常の時空図 (t, x) を描く
2. 光線 (t = \pm x) を引く
3. 光座標 (u = t-x,\ v = t+x) を導入
4. (U = \arctan(u),\ V = \arctan(v)) を使う
5. ペンローズ図を描く

---

## ✏️ 手で描く手順（実践）

① まず普通の図を書く

```
   t
   ↑
   |
   O──→ x
```

② 光（45°）を書く
③ ダイヤ型に潰す

---

## ✔️ 正解の形

```
        i⁺
       / \
      /   \
     /     \
  𝓘⁻       𝓘⁺
     \     /
      \   /
       \ /
        i⁻
```

---

## 💡チェックポイント

* 光が45°になっているか？
* 図が「ダイヤ型」になっているか？

---

# 🟡 レベル2：リンドラー時空（加速系）

（地平線の理解）

対象：リンドラー時空

---

## 🔹問題2

一様加速する観測者の世界線

[
x^2 - t^2 = \text{const}
]

を描き、その観測者のペンローズ図上で：

* 見えない領域（地平線）を示せ

---

## ✏️ 手順

① ミンコフスキーのペンローズ図を描く
② 双曲線（加速運動）を書く
③ 光線を引く

---

## ✔️ 見るべき結果

* 観測者には見えない領域がある
* 地平線が「45°の線」で現れる

👉 これがブラックホールの地平線の予習

---

# 🔴 レベル3：シュワルツシルト時空（本番）

（ブラックホール）

対象：シュワルツシルト時空

---

## 🔹問題3

以下を満たすペンローズ図を描け：

* 外部領域
* 事象の地平線
* 特異点

---

## ✏️ ヒント手順

① ミンコフスキー図をベースにする
② 「地平線」を追加（45°線）
③ 「特異点」を上に置く

---

## ✔️ 正解のイメージ

```
        特異点
       ───────
        /   \
       /     \
      /       \
  外部   |H|   外部
      \       /
       \     /
        \   /
         \ /
```

---

## 💡理解ポイント

* 内部では「未来＝特異点」
* 外に出る未来は存在しない

👉 これが
「未来が特異点に向く」
の図的意味
