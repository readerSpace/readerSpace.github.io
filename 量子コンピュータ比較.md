古典ビット回路と量子ビット回路を並べて、ゲートを追加→実行→結果を見る、という1ファイルの教材用シミュレーションとして作ります。量子側は1量子ビット＋簡易的な測定確率から始める形にします

import React, { useMemo, useState } from "react";
import { motion } from "framer-motion";
import { Button } from "@/components/ui/button";
import { Card, CardContent } from "@/components/ui/card";
import { Plus, RotateCcw, Play, Trash2, Cpu, Atom } from "lucide-react";

const classicalGates = [
  { name: "NOT", desc: "0⇄1", fn: (b) => 1 - b },
  { name: "BUF", desc: "そのまま", fn: (b) => b },
];

const quantumGates = [
  {
    name: "X",
    desc: "|0〉⇄|1〉",
    matrix: [
      [0, 1],
      [1, 0],
    ],
  },
  {
    name: "H",
    desc: "重ね合わせ",
    matrix: [
      [1 / Math.sqrt(2), 1 / Math.sqrt(2)],
      [1 / Math.sqrt(2), -1 / Math.sqrt(2)],
    ],
  },
  {
    name: "Z",
    desc: "位相反転",
    matrix: [
      [1, 0],
      [0, -1],
    ],
  },
];

function applyQuantumGate(state, matrix) {
  const [a, b] = state;
  return [
    matrix[0][0] * a + matrix[0][1] * b,
    matrix[1][0] * a + matrix[1][1] * b,
  ];
}

function clampPercent(x) {
  return Math.max(0, Math.min(100, x));
}

function fmt(x) {
  if (Math.abs(x) < 1e-10) return "0.00";
  return x.toFixed(2);
}

function GateChip({ gate, onRemove }) {
  return (
    <motion.div
      layout
      initial={{ opacity: 0, y: -8 }}
      animate={{ opacity: 1, y: 0 }}
      exit={{ opacity: 0 }}
      className="flex items-center gap-2 rounded-2xl border bg-white px-3 py-2 shadow-sm"
    >
      <span className="font-bold">{gate}</span>
      <button onClick={onRemove} className="text-gray-400 hover:text-red-500">
        ×
      </button>
    </motion.div>
  );
}

function ProbabilityBar({ label, value }) {
  return (
    <div className="space-y-1">
      <div className="flex justify-between text-sm">
        <span>{label}</span>
        <span>{value.toFixed(1)}%</span>
      </div>
      <div className="h-4 overflow-hidden rounded-full bg-gray-200">
        <motion.div
          className="h-full rounded-full bg-gray-900"
          initial={{ width: 0 }}
          animate={{ width: `${clampPercent(value)}%` }}
          transition={{ duration: 0.35 }}
        />
      </div>
    </div>
  );
}

export default function ClassicalQuantumGateSimulator() {
  const [classicalInput, setClassicalInput] = useState(0);
  const [classicalCircuit, setClassicalCircuit] = useState(["NOT"]);
  const [quantumInput, setQuantumInput] = useState("0");
  const [quantumCircuit, setQuantumCircuit] = useState(["H"]);
  const [measurements, setMeasurements] = useState([]);

  const classicalResult = useMemo(() => {
    let bit = classicalInput;
    const steps = [{ label: "入力", value: bit }];
    for (const gateName of classicalCircuit) {
      const gate = classicalGates.find((g) => g.name === gateName);
      bit = gate.fn(bit);
      steps.push({ label: gateName, value: bit });
    }
    return { bit, steps };
  }, [classicalInput, classicalCircuit]);

  const quantumResult = useMemo(() => {
    let state = quantumInput === "0" ? [1, 0] : [0, 1];
    const steps = [{ label: `|${quantumInput}〉`, state }];
    for (const gateName of quantumCircuit) {
      const gate = quantumGates.find((g) => g.name === gateName);
      state = applyQuantumGate(state, gate.matrix);
      steps.push({ label: gateName, state });
    }
    const p0 = state[0] * state[0] * 100;
    const p1 = state[1] * state[1] * 100;
    return { state, p0, p1, steps };
  }, [quantumInput, quantumCircuit]);

  const addClassicalGate = (name) => setClassicalCircuit((prev) => [...prev, name]);
  const addQuantumGate = (name) => setQuantumCircuit((prev) => [...prev, name]);

  const measureQuantum = () => {
    const r = Math.random() * 100;
    const result = r < quantumResult.p0 ? "0" : "1";
    setMeasurements((prev) => [result, ...prev].slice(0, 20));
  };

  const counts = measurements.reduce(
    (acc, m) => {
      acc[m] += 1;
      return acc;
    },
    { 0: 0, 1: 0 }
  );

  return (
    <div className="min-h-screen bg-gradient-to-br from-slate-50 to-slate-200 p-4 text-slate-900">
      <div className="mx-auto max-w-6xl space-y-5">
        <header className="space-y-2 py-4">
          <h1 className="text-3xl font-bold tracking-tight md:text-4xl">
            古典コンピュータ vs 量子コンピュータ 回路シミュレーター
          </h1>
          <p className="max-w-3xl text-slate-600">
            左は普通のビット、右は量子ビット。ゲートを追加して、結果がどう変わるか試せます。
          </p>
        </header>

        <div className="grid gap-5 lg:grid-cols-2">
          <Card className="rounded-3xl shadow-lg">
            <CardContent className="space-y-5 p-5">
              <div className="flex items-center gap-2">
                <Cpu className="h-6 w-6" />
                <h2 className="text-2xl font-bold">古典回路</h2>
              </div>

              <div className="rounded-2xl bg-slate-100 p-4">
                <p className="mb-2 font-semibold">入力ビット</p>
                <div className="flex gap-2">
                  {[0, 1].map((b) => (
                    <Button
                      key={b}
                      variant={classicalInput === b ? "default" : "outline"}
                      onClick={() => setClassicalInput(b)}
                      className="rounded-2xl"
                    >
                      {b}
                    </Button>
                  ))}
                </div>
              </div>

              <div>
                <p className="mb-2 font-semibold">追加できるゲート</p>
                <div className="flex flex-wrap gap-2">
                  {classicalGates.map((g) => (
                    <Button key={g.name} onClick={() => addClassicalGate(g.name)} className="rounded-2xl">
                      <Plus className="mr-1 h-4 w-4" /> {g.name} <span className="ml-2 text-xs opacity-70">{g.desc}</span>
                    </Button>
                  ))}
                </div>
              </div>

              <div className="rounded-2xl border bg-white p-4">
                <div className="mb-3 flex items-center justify-between">
                  <p className="font-semibold">回路</p>
                  <Button variant="ghost" size="sm" onClick={() => setClassicalCircuit([])}>
                    <Trash2 className="mr-1 h-4 w-4" /> 消去
                  </Button>
                </div>
                <div className="flex min-h-14 flex-wrap items-center gap-2">
                  <div className="rounded-2xl bg-slate-900 px-3 py-2 font-bold text-white">{classicalInput}</div>
                  <div className="text-slate-400">→</div>
                  {classicalCircuit.length === 0 && <span className="text-slate-400">ゲートなし</span>}
                  {classicalCircuit.map((g, i) => (
                    <React.Fragment key={`${g}-${i}`}>
                      <GateChip
                        gate={g}
                        onRemove={() => setClassicalCircuit((prev) => prev.filter((_, idx) => idx !== i))}
                      />
                      <div className="text-slate-400">→</div>
                    </React.Fragment>
                  ))}
                  <div className="rounded-2xl bg-emerald-600 px-3 py-2 font-bold text-white">
                    {classicalResult.bit}
                  </div>
                </div>
              </div>

              <div className="rounded-2xl bg-slate-900 p-5 text-white">
                <p className="text-sm opacity-70">出力</p>
                <p className="text-5xl font-black">{classicalResult.bit}</p>
                <p className="mt-2 text-sm opacity-80">
                  古典ビットは、測定しなくても結果が1つに決まります。
                </p>
              </div>
            </CardContent>
          </Card>

          <Card className="rounded-3xl shadow-lg">
            <CardContent className="space-y-5 p-5">
              <div className="flex items-center gap-2">
                <Atom className="h-6 w-6" />
                <h2 className="text-2xl font-bold">量子回路</h2>
              </div>

              <div className="rounded-2xl bg-slate-100 p-4">
                <p className="mb-2 font-semibold">入力量子ビット</p>
                <div className="flex gap-2">
                  {["0", "1"].map((b) => (
                    <Button
                      key={b}
                      variant={quantumInput === b ? "default" : "outline"}
                      onClick={() => setQuantumInput(b)}
                      className="rounded-2xl"
                    >
                      |{b}〉
                    </Button>
                  ))}
                </div>
              </div>

              <div>
                <p className="mb-2 font-semibold">追加できるゲート</p>
                <div className="flex flex-wrap gap-2">
                  {quantumGates.map((g) => (
                    <Button key={g.name} onClick={() => addQuantumGate(g.name)} className="rounded-2xl">
                      <Plus className="mr-1 h-4 w-4" /> {g.name} <span className="ml-2 text-xs opacity-70">{g.desc}</span>
                    </Button>
                  ))}
                </div>
              </div>

              <div className="rounded-2xl border bg-white p-4">
                <div className="mb-3 flex items-center justify-between">
                  <p className="font-semibold">回路</p>
                  <Button variant="ghost" size="sm" onClick={() => setQuantumCircuit([])}>
                    <Trash2 className="mr-1 h-4 w-4" /> 消去
                  </Button>
                </div>
                <div className="flex min-h-14 flex-wrap items-center gap-2">
                  <div className="rounded-2xl bg-slate-900 px-3 py-2 font-bold text-white">|{quantumInput}〉</div>
                  <div className="text-slate-400">→</div>
                  {quantumCircuit.length === 0 && <span className="text-slate-400">ゲートなし</span>}
                  {quantumCircuit.map((g, i) => (
                    <React.Fragment key={`${g}-${i}`}>
                      <GateChip
                        gate={g}
                        onRemove={() => setQuantumCircuit((prev) => prev.filter((_, idx) => idx !== i))}
                      />
                      <div className="text-slate-400">→</div>
                    </React.Fragment>
                  ))}
                  <div className="rounded-2xl bg-indigo-600 px-3 py-2 font-bold text-white">測定</div>
                </div>
              </div>

              <div className="grid gap-4 rounded-2xl bg-slate-900 p-5 text-white md:grid-cols-2">
                <div>
                  <p className="text-sm opacity-70">現在の量子状態</p>
                  <p className="mt-2 text-lg font-bold">
                    {fmt(quantumResult.state[0])}|0〉 {quantumResult.state[1] >= 0 ? "+" : "−"} {fmt(Math.abs(quantumResult.state[1]))}|1〉
                  </p>
                  <p className="mt-2 text-xs opacity-70">
                    Hゲートを使うと、0と1が確率的に出る状態になります。
                  </p>
                </div>
                <div className="space-y-3">
                  <ProbabilityBar label="測定で 0 が出る確率" value={quantumResult.p0} />
                  <ProbabilityBar label="測定で 1 が出る確率" value={quantumResult.p1} />
                </div>
              </div>

              <div className="rounded-2xl border bg-white p-4">
                <div className="flex flex-wrap items-center gap-2">
                  <Button onClick={measureQuantum} className="rounded-2xl">
                    <Play className="mr-2 h-4 w-4" /> 測定してみる
                  </Button>
                  <Button variant="outline" onClick={() => setMeasurements([])} className="rounded-2xl">
                    <RotateCcw className="mr-2 h-4 w-4" /> 測定履歴リセット
                  </Button>
                </div>
                <div className="mt-4">
                  <p className="font-semibold">最近の測定結果</p>
                  <div className="mt-2 flex min-h-10 flex-wrap gap-2">
                    {measurements.length === 0 && <span className="text-slate-400">まだ測定していません</span>}
                    {measurements.map((m, i) => (
                      <motion.div
                        key={`${m}-${i}`}
                        initial={{ scale: 0.7, opacity: 0 }}
                        animate={{ scale: 1, opacity: 1 }}
                        className="rounded-xl bg-indigo-100 px-3 py-1 font-bold text-indigo-900"
                      >
                        {m}
                      </motion.div>
                    ))}
                  </div>
                  {measurements.length > 0 && (
                    <p className="mt-3 text-sm text-slate-600">
                      0: {counts[0]}回 / 1: {counts[1]}回。確率なので、毎回同じ結果になるとは限りません。
                    </p>
                  )}
                </div>
              </div>
            </CardContent>
          </Card>
        </div>

        <Card className="rounded-3xl shadow-lg">
          <CardContent className="space-y-3 p-5">
            <h2 className="text-xl font-bold">観察ポイント</h2>
            <div className="grid gap-3 md:grid-cols-3">
              <div className="rounded-2xl bg-white p-4 shadow-sm">
                <p className="font-bold">古典ゲート</p>
                <p className="text-sm text-slate-600">ビットは常に0か1。NOTを2回かけると元に戻ります。</p>
              </div>
              <div className="rounded-2xl bg-white p-4 shadow-sm">
                <p className="font-bold">Hゲート</p>
                <p className="text-sm text-slate-600">|0〉にHをかけると、0と1がほぼ半々で出る状態になります。</p>
              </div>
              <div className="rounded-2xl bg-white p-4 shadow-sm">
                <p className="font-bold">Zゲート</p>
                <p className="text-sm text-slate-600">確率は変えず、符号だけ変えます。Hと組み合わせると結果に影響します。</p>
              </div>
            </div>
          </CardContent>
        </Card>
      </div>
    </div>
  );
}
左が古典回路、右が量子回路で、ゲート追加・削除・測定を試せます。
